# Docker命令备忘

```
## 列出Docker CLI命令
docker
docker container --help

## 显示Docker版本和信息
docker --version
docker version
docker info

## 运行Docker镜像
docker run hello-world
docker run <image-name>

## 列出Docker镜像
docker image ls

## 列出Docker容器(运行中，全部，安静模式中所有的)
docker container ls
docker container ls --all
docker container ls -aq
```
